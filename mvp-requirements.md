# Image Background Remover MVP 需求文档

## 1. 产品概述

**产品名称**：待定（可叫 ClipDrop China / RemoveCN / 图背景）

**一句话介绍**：一键移除图片背景的在线工具

**核心价值**：让用户无需 PS 技能，3秒内获得透明背景图片

**目标用户**：
- 电商卖家（淘宝/拼多多/亚马逊）
- 设计师 / 运营 / 摄影师
- 需要快速抠图的个人用户

---

## 2. 用户故事

| 角色 | 场景 | 需求 |
|---|---|---|
| 电商运营 | 上传商品图 | 快速去掉白底，换成透明背景 |
| 设计师 | 做素材 | 提取产品主体，用于海报设计 |
| 摄影师 | 修图 | 换背景、做合成 |
| 普通人 | 日常 | 证件照换底色、做表情包 |

## 3. 功能范围

### MVP 必须功能

- [x] **图片上传**：支持拖拽 + 点击上传
- [x] **格式支持**：JPG / PNG / WebP，最大 10MB
- [x] **背景移除**：调用 Remove.bg API 完成抠图
- [x] **结果预览**：实时展示原图 vs 效果图（左右/上下对比）
- [x] **免费下载**：导出 PNG（透明背景）
- [x] **免费次数**：注册用户每天 5 次免费（未注册 1 次）

### 后续版本功能（不在 MVP 范围）

- [ ] 批量处理（一次上传多张）
- [ ] 背景替换（换成纯色/自定义图片）
- [ ] 高清原图下载（付费功能）
- [ ] API 开放（给其他开发者）
- [ ] Chrome 插件

## 5. 技术方案

### 技术栈

| 层级 | 技术选型 |
|---|---|
| 前端 | React 18 + TailwindCSS（单页应用） |
| 后端 | Cloudflare Workers（边缘函数，内存处理） |
| 图片存储 | 纯内存，不持久化（隐私优先） |
| AI API | Remove.bg API（$0.09/图） |
| 部署 | Cloudflare（前端+后端一体化） |

### 核心 API 设计

```
POST /api/remove-bg
  Body: FormData { image: File }
  Headers: Authorization: Bearer <token>
  Response: { success: true, imageUrl: "https://...", remaining: 4 }

GET /api/user/remaining
  Headers: Authorization: Bearer <token>
  Response: { remaining: 4, total: 5 }
```

### 数据流

```
用户上传 → 前端预览 → POST /api/remove-bg
         → Cloudflare Worker 接收（内存 Buffer）
         → 调用 Remove.bg API
         → 直接返回 PNG 给前端
         → 不存储，内存处理完即释放
```

**Cloudflare Workers 优势**：
- 全球边缘部署，延迟低
- 按请求计费，免费额度 100k 请求/天
- 原生支持 WebAssembly，可跑 AI 模型（后期）

## 6. 非功能需求

### 性能

- 图片处理时间：< 5秒（取决于图片大小和网络）
- 同时并发：MVP 暂不限制，后期加队列

### 隐私

- 图片不持久化存储，处理完即删
- 不收集用户人脸/隐私数据

### 合规

- Remove.bg 需遵守其 TOS
- 前端需声明"图片仅用于背景移除"
